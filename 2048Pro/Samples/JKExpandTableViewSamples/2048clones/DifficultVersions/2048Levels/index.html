How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

Difficulty settings:

  • Easy: same as original game, new tiles are generated randomly
  • Medium: computer will place the new tile in an annoying place 25% of the time
  • Hard: computer will place the new tile in an annoying place 50% of the time
  • Impossible: computer will always place the tile in an annoying place.

Using: this section will determine which values can be generated on the grid by the computer. The default is to generate both 2s and 4s. Switch it to 2s only if you want an easier game, or increase it for fun.
